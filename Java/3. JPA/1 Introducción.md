# 1. Introducción

## ¿Qué es JPA?

JPA → Java Persistance API - API de persistencia de Java

Es una herramienta que permite a los desarrolladores de software trabajar con bases de datos relacionales utilizando el lenguaje de programación Java.

## Utilidad de JPA

Con JPA, los desarrolladores pueden mapear objetos Java a tablas de bases de datos y realizar operaciones de lectura y escritura en ellas utilizando un conjunto de clases y métodos definidos por la API.

---

## ¿Qué es un ORM?

ORM → Object Relational Mapping - Asignación relacional de objetos (MyBatis, Hibernate)

Es una técnica de programación que permite relacionar objetos de un lenguaje de programación con tablas de una base de datos relacional.

## Utilidad de ORM

Es una herramienta de programación que permite interactuar con bases de datos a través de objetos en lugar de escribir instrucciones SQL. El uso de un ORM facilita el desarrollo de aplicaciones y ayuda a prevenir errores comunes en el manejo de bases de datos.

# 2. Ejecución en dispositivo físico

Para ejecutar nuestra aplicación en nuestro teléfono tendremos que ir a ajustes→ acerca de→ y buscar el numero de compilación.

Una vez lo encuentres:

1. le das 7 veces para activar el modo developer.
2. Vamos a las opciones de desarrollo.
3. Buscamos una la opción la de depuración por USB y la activamos.

Ahora conectamos el cable al pc y ejecutamos la app (le tendremos que indicar que queremos usar el **telefono fisico**)
